# Chess Engine

A personal chess engine written in C++.

This project started as a passion project and an opportunity to build something challenging from the ground up. As someone who enjoys both chess and software engineering, I wanted to better understand the mechanics behind how a chess engine represents positions, generates moves, evaluates games, and makes decisions.

The goal isn't to compete with top engines, but to create something I'm proud of while improving my understanding of system design, performance optimization, and modern C++.

## Features

- FEN Move parsing
- Legal move validation
- Position evaluation (check/mate)
- Game state management
- Correctness testing of the different features
- Continuous performance improvements

## Status

Work in progress. New features, optimizations, and refactors are added as the project evolves.

## For the Future
I'd love to eventually add the following:
- Minmax or other AI algorithm for a vs computer mode
- Bitboard representations of the position and moves
- Improve efficiency of different sections of the algorithm (use profiling tools to compare implementations)
- Multiplayer mode
